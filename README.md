# Sierpinski Carpet
Sierpinski carpet fractal written in c++ while I was on 10h flight to USA.

# Dependencies
- [SFML](http://www.sfml-dev.org)

# Installation
```
mkdir build
cd build
cmake ..
make -j 8
../bin/Sierpinski_Carpet
```
